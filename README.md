# 🐍 Beginner Level Python Projects

A collection of beginner Python projects, each rebuilt with a proper GUI (instead of the command line).

Each project lives in its own folder with its own code, its own `requirements.txt`, and its own README. Different projects may use different GUI libraries depending on what suits them best.

## How to run any project

```bash
cd <project-folder-name>
pip install -r requirements.txt
python <main_file>.py
```
